## 📞 چه کنم اگر؟

### ❓ نمی‌دانم کجا شروع کنم
```
→ بخوان: README_FA.md
```

### ❓ می‌خوام سریع استفاده کنم
```
→ بخوان: QUICK_START.md (فقط اول)
```

### ❓ نمی‌فهمم چگونه کار می‌کند
```
→ بخوان: ARCHITECTURE.md + PLUGIN_DOCUMENTATION.md
```

### ❓ می‌خوام پلاگین را توسعه دهم
```
→ بخوان: ARCHITECTURE.md + PLUGIN_DOCUMENTATION.md + کد
```

### ❓ خطای غریبی می‌گیرم
```
→ بخوان: QUICK_START.md (بخش ترابل‌شوتینگ)
→ یا: PLUGIN_DOCUMENTATION.md (بخش API Reference)
```

### ❓ می‌خوام نمونه‌ی JSON درست بدهم
```
→ کپی کن: EXAMPLE_COMPLETE_EXPORT.json
→ تعدیل کن برای پروژه‌ات
```
