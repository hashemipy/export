# شروع سریع

## مرحله ۱: درک مشکل اصلی

**مشکل قدیمی:**
```
سایت مبدا ──(بدون دسته‌بندی و ویژگی)──> سایت مقصد
                                       ❌ محصول ساده ایجاد می‌شود
                                       ❌ متغیرها نیستند
                                       ❌ موجودی اشتباه
```

**حل جدید:**
```
سایت مبدا ──────> دانلود کامل ──────> پردازش ──────> اپلود ترتیبی ──────> سایت مقصد
             (تمام وابستگی‌ها)    (تنظیف)    (۱.دسته‌بندی)         ✅ محصول متغیر
                                              (۲.ویژگی)           ✅ تمام متغیرها
                                              (۳.محصول)          ✅ موجودی صحیح
                                              (۴.متغیرها)
```

## مرحله ۳: چگونه کار می‌کند؟

### مثال: انتقال محصول "تیشرت"

#### سایت مبدا دارای:
```
📦 محصول: تیشرت (variable)
  ├─ 🎨 ویژگی "رنگ": قرمز، آبی، سبز
  ├─ 📏 ویژگی "سایز": 40، 45، 50، 55، 60، 65
  ├─ 📂 دسته: "لباس" → "تیشرت"
  └─ 📦 10 متغیر مختلف
      ├─ SKU: 1-RED-40 (موجودی: 5)
      ├─ SKU: 1-RED-45 (موجودی: 3)
      ├─ SKU: 1-BLUE-40 (موجودی: 7)
      └─ ...
```

#### مرحله دانلود:
```javascript
downloader.downloadComplete()
↓
{
  products: [محصول تیشرت + 10 متغیر],
  categories: [دسته "لباس", دسته "تیشرت"],
  attributes: [ویژگی "رنگ" (3 مقدار), ویژگی "سایز" (6 مقدار)]
}
```

#### مرحله پردازش:
```javascript
processor.processAll()
✓ تکراری‌ها حذف شد
✓ دسته‌های مادر ابتدا
✓ SKU تکراری تنظیم شد
✓ موجودی کل = 5+3+7+... = 35
✓ بررسی کامل انجام شد
```

#### مرحله اپلود:
```
۱️⃣  ایجاد دسته‌بندی "لباس" (مادر)
   ↓
۲️⃣  ایجاد دسته‌بندی "تیشرت" (فرزند)
   ↓
۳️⃣  ایجاد ویژگی "رنگ" با مقادیر: قرمز، آبی، سبز
   ↓
۴️⃣  ایجاد ویژگی "سایز" با مقادیر: 40، 45، 50، 55، 60، 65
   ↓
۵️⃣  ایجاد محصول "تیشرت" (type: variable)
   ↓
۶️⃣  ایجاد 10 متغیر:
      SKU: 1-RED-40 (موجودی: 5)
      SKU: 1-RED-45 (موجودی: 3)
      SKU: 1-BLUE-40 (موجودی: 7)
      ...
```

#### نتیجه:
```
✅ سایت مقصد = سایت مبدا (کپی دقیق)
✅ تمام دسته‌بندی‌ها
✅ تمام ویژگی‌ها
✅ تمام متغیرها
✅ موجودی دقیق برای هر متغیر
```

## مرحله ۴: استفاده واقعی

### گزینه الف: رابط کاربری (راحت‌ترین)

```
1. وب‌سایت را باز کنید
2. تب "انتقال کامل" را انتخاب کنید
3. آدرس سایت مبدا وارد کنید
4. آدرس سایت مقصد وارد کنید
5. "شروع انتقال" کلیک کنید
6. تماشا کنید! ✨
```

### گزینه ب: دانلود و سپس اپلود

```
1. تب "دانلود محصولات"
   ↓ دانلود فایل JSON
2. تب "اپلود به سایت مقصد"
   ↓ اپلود فایل JSON
```

### گزینه ج: برنامه‌نویسی (پیشرفته)

```typescript
// کد مستقیم در پروژه شما
import { ProductMigrationManager } from '@/lib/plugins';

const manager = new ProductMigrationManager(
  'https://source.com',
  'https://dest.com',
  'your-api-key'
);

const result = await manager.migrateComplete();
console.log('✓ انتقال انجام شد:', result);
```

## مرحله ۵: چک‌لیست قبل و بعد

### ✅ قبل از انتقال
- [ ] آدرس سایت مبدا صحیح است
- [ ] آدرس سایت مقصد صحیح است
- [ ] محصولات در سایت مبدا ویژگی‌ داشته باشند
- [ ] متغیرها کامل هستند
- [ ] موجودی درست است

### ✅ بعد از انتقال
- [ ] محصول در سایت مقصد متغیر است (نه ساده)
- [ ] تمام ویژگی‌ها ایجاد شدند
- [ ] تمام متغیرها موجود هستند
- [ ] موجودی هر متغیر صحیح است
- [ ] دسته‌بندی‌های مادر و فرزند درست هستند

---

## مرحله ۶: حل مشکلات

### ❌ محصول ساده ایجاد شد
**علت:** ویژگی‌ها ایجاد نشدند
**حل:** اپلود دوباره کنید

### ❌ متغیر‌ها ایجاد نشدند
**علت:** محصول ابتدا ایجاد نشد
**حل:** قبل از متغیرها، محصول باید ایجاد شود

### ❌ موجودی اشتباه است
**علت:** SKU تکراری
**حل:** فایل JSON را بررسی کنید - هر متغیر SKU منحصر به فرد دارد

### ❌ ویژگی‌های رنگ و سایز نمایش نمی‌دهند
**علت:** visible: false
**حل:** در فایل JSON، visible: true قرار دهید

## مرحله ۷: فایل JSON صحیح

```json
{
  "products": [
    {
      "name": "محصول",
      "type": "variable",  // ✅ الزامی: "variable"
      "sku": "123",        // ✅ الزامی
      "attributes": {      // ✅ الزامی برای variable
        "color": {
          "name": "رنگ",
          "values": [
            { "name": "قرمز", "slug": "red" }
          ]
        }
      },
      "variations": [      // ✅ الزامی برای variable
        {
          "sku": "123-red-40",  // ✅ منحصر به فرد
          "attributes": {
            "color": "red"
          },
          "stock_quantity": 10
        }
      ]
    }
  ]
}
```

## مرحله ۸: خطوات بعدی

1. **سایت خود را وارد کنید:**

   - یک محصول متغیر انتخاب کنید
   - انتقال دهید
   - بررسی کنید

3. **اگر موفق بود، همه را انتقال دهید**
