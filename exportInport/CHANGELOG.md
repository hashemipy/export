## تغییرات اخیر - نسخه 1.1.0

### ✨ ویژگی‌های جدید

#### 1. **جدول انتخاب‌پذیر محصولات**
- نمایش تمام محصولات در یک جدول حرفه‌ای
- ستون‌های مهم: نام، SKU، قیمت، موجودی، نوع
- انتخاب/عدم‌انتخاب محصولات برای دانلود

#### 2. **دانلود انتخاب‌شده**
- **فقط محصولات انتخاب‌شده** دانلود می‌شوند
- تعداد محصولات انتخاب‌شده نمایش داده می‌شود
- دکمه "انتخاب همه" برای سهولت

#### 3. **فیلترهای جدید**
- **همه**: تمام محصولات
- **ساده**: فقط محصولات ساده
- **متغیر**: فقط محصولات متغیر

#### 4. **رابط کاربری بهتر**
- طراحی جانبی: جدول + تنظیمات
- نمایش زیبای جدول محصولات
- پیغام‌های وضعیت واضح و فوری

## دستورات AJAX جدید

### 1. `pie_get_products_list` (جدید)
**هدف:** بارگذاری لیست محصولات برای نمایش در جدول

**پارامترها:**
- `action`: `pie_get_products_list`
- `nonce`: نامه امنیتی

**پاسخ:**
```json
{
  "success": true,
  "data": [
    {
      "id": 123,
      "name": "نام محصول",
      "sku": "SKU123",
      "price": "150,000 تومان",
      "stock_quantity": 10,
      "type": "simple"
    }
  ]
}
```

### 2. `pie_download_products` (اصلاح‌شده)
**تغییر:** به‌جای `simple` و `variable`، حالا `product_ids` قبول می‌کند

**پارامترهای قدیم:**
```javascript
data: {
    action: 'pie_download_products',
    simple: 1,
    variable: 1
}
```

**پارامترهای جدید:**
```javascript
data: {
    action: 'pie_download_products',
    product_ids: "123,456,789",
    format: "json"
}
```

## تغییرات کد

### فایل اصلی: `product-import-export.php`

#### متدهای جدید:
1. `handle_get_products_list()` - بارگذاری لیست محصولات
2. `get_selected_products_data()` - دریافت داده‌های محصولات انتخاب‌شده

#### متدهای حذف‌شده:
1. `get_products_data()` - جایگزین شد با `get_selected_products_data()`

#### متدهای اصلاح‌شده:
1. `render_page()` - رابط کاربری جدید
2. `handle_download()` - دانلود محصولات انتخاب‌شده
3. `__construct()` - اضافه کردن AJAX action جدید

#### جاوااسکریپت جدید:
```javascript
- loadProductsList()        // بارگذاری لیست
- renderProductsTable()     // رندر جدول
- updateSelectedCount()     // به‌روزرسانی تعداد
- فیلتر عملکرد
- انتخاب checkbox
```

## راهنمای مهاجرت

### اگر کد قدیم استفاده می‌کنید:

**قبل:**
```php
$products = $this->get_products_data(1, 1);
```

```php
$product_ids = [123, 456, 789];
$products = $this->get_selected_products_data($product_ids);
```
