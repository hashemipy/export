# پلاگین Import/Export - نسخه 2.0.0 اصلاح شده

## ✅ اصلاحات انجام شده

### 1. **دسته‌بندی‌ها (Categories)**
- استفاده از API استاندارد WooCommerce
- تابع `sync_category()` برای ایجاد/دریافت دسته‌بندی
- پشتیبانی کامل والدین (parent categories)
- بدون خطای WooCommerce

### 2. **ویژگی‌ها (Attributes)**
- استفاده از `wc_create_attribute()` (تابع رسمی WooCommerce)
- تابع `sync_attribute()` برای ایجاد ویژگی
- ایجاد خودکار مقادیر ویژگی (attribute terms)
- بدون خطای taxonomy

### 3. **متغیرات (Variations)**
- استفاده از کلاس `WC_Product_Variation` (صحیح)
- تابع `create_variation()` برای ایجاد متغیر
- اختصاص صحیح ویژگی‌ها
- قیمت و موجودی جداگانه برای هر متغیر

### 4. **عکس‌ها (Images)**
- دانلود از URL (نه فایل کامل)
- تابع `download_image()` برای دانلود عکس
- اتصال خودکار به محصول
- عکس اصلی + گالری

### 5. **سرعت و ایمنی**
- بدون تکرار درخواست‌ها
- بررسی صحیح اطلاعات
- مدیریت خطا
- logging خودکار

## 🚀 مراحل استفاده

### دانلود محصولات:
1. بروید: WooCommerce → Import/Export
2. محصول را انتخاب کنید
3. فرمت JSON انتخاب کنید
4. دانلود کنید

### آپلود محصولات:
1. فایل JSON را انتخاب کنید
2. آپلود کنید
3. خودکار:
   - ✓ دسته‌بندی ایجاد
   - ✓ ویژگی ایجاد
   - ✓ محصول ایجاد
   - ✓ متغیرات ایجاد
   - ✓ عکس‌ها دانلود
4. گزارش دریافت کنید

## 🔧 توابع اصلی

| تابع | توضیح |
|------|--------|
| `sync_category()` | ایجاد/دریافت دسته‌بندی |
| `sync_attribute()` | ایجاد/دریافت ویژگی |
| `create_variation()` | ایجاد متغیر |
| `download_image()` | دانلود عکس از URL |
| `import_products()` | آپلود محصولات |
