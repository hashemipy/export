# سیستم بررسی و تأیید فایل‌ها

## ✨ ویژگی جدید

حالا قبل از آپلود، فایل شما بررسی می‌شود:

1. **بررسی ساختار JSON** ✓
2. **بررسی نام محصولات** ✓
3. **بررسی قیمت** ✓
4. **بررسی ویژگی‌ها (برای متغیر)** ✓
5. **بررسی متغیرات** ✓
6. **نمایش تیک‌های سبز برای معتبر** ✓
7. **نمایش خطاها برای نامعتبر** ✓

## مراحل استفاده

### 1. فایل را انتخاب کنید
- دکمه "انتخاب فایل" را کلیک کنید
- فایل JSON یا CSV را انتخاب کنید

### 2. بررسی فایل
- دکمه "✓ بررسی فایل" را کلیک کنید
- نتایج نمایش داده می‌شود

### 3. بررسی نتایج

**اگر معتبر است:**
```
✓ 2/2 محصول معتبر

✓ فایل معتبر است! می‌توانید اپلود کنید
```
→ دکمه "آپلود محصولات" فعال می‌شود

**اگر مشکل دارد:**
```
❌ خطاها (3):
• محصول "تیشرت": ویژگی‌ها موجود نیست
• محصول "شلوارک" - متغیر 1: قیمت موجود نیست
• محصول "ست": نام موجود نیست

✗ فایل دارای خطا است. لطفا اصلاح کنید
```
→ فایل را اصلاح کنید و دوباره بررسی کنید

### 4. آپلود
- دکمه "→ آپلود محصولات" را کلیک کنید
- محصولات آپلود می‌شوند

## انواع خطاها

### نام موجود نیست
```
محصول 1: نام موجود نیست
```
**حل:** نام محصول را اضافه کنید

### قیمت موجود نیست
```
محصول "تیشرت": قیمت موجود نیست
```
**حل:** برای محصولات ساده، قیمت الزامی است

### ویژگی‌ها موجود نیست (متغیر)
```
محصول "شلوارک": ویژگی‌ها موجود نیست
```
**حل:** برای محصولات متغیر، attributes الزامی است

### متغیرات موجود نیست (متغیر)
```
محصول "کت": متغیرات موجود نیست
```
**حل:** برای محصولات متغیر، variations الزامی است

### ویژگی‌های متغیر خالی
```
محصول "شلوارک" - متغیر 1: ویژگی‌های متغیر خالی است
```
**حل:** هر متغیر باید attributes داشته باشد

## نمونه JSON صحیح

فایل `products-correct-example.json` را ببینید برای نمونه کامل.

### ساختار کلی:

```json
[
  {
    "name": "نام محصول",
    "sku": "کد محصول",
    "price": "10000",
    "stock_quantity": 5,
    "type": "simple",
    "categories": [...],
    "image_urls": [...],
    "attributes": {},
    "variations": []
  }
]
```

### محصول متغیر (حتمی):

```json
{
  "type": "variable",
  "attributes": {
    "color": {
      "values": [
        {"name": "سفید", "slug": "white"},
        {"name": "سیاه", "slug": "black"}
      ]
    },
    "size": {
      "values": [
        {"name": "S", "slug": "s"},
        {"name": "M", "slug": "m"}
      ]
    }
  },
  "variations": [
    {
      "sku": "P-W-S",
      "price": "10000",
      "stock_quantity": 5,
      "attributes": {
        "color": "سفید",
        "size": "S"
      }
    }
  ]
}
```

## هشدارها (Warnings)

هشدارها مشکلات غیرحتمی هستند:

```
⚠️ هشدارها (1):
• محصول "تیشرت" - متغیر 1: قیمت موجود نیست
```

→ می‌توانید با هشدار آپلود کنید

## آپلود موفق

بعد از آپلود موفق:

```
✓ محصولات: 2 | دسته‌بندی‌ها: 1 | ویژگی‌ها: 2 | متغیرات: 6 | عکس‌ها: 2
```
