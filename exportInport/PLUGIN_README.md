# پلاگین محصول Import/Export

یک پلاگین سریع و ساده برای دانلود و آپلود محصولات WooCommerce

## ویژگی‌ها

### 📥 دانلود محصولات
- دانلود به صورت **JSON** یا **CSV**
- فیلتر محصولات ساده و متغیر
- شامل تمام اطلاعات: نام، SKU، قیمت، موجودی، دسته‌بندی، عکس‌ها
- برای محصولات متغیر: ویژگی‌ها و متغیرهای تفصیلی

### 📤 آپلود محصولات
- آپلود از فایل JSON یا CSV
- نشان دادن پیشرفت آپلود
- تشخیص خطاها
- ایجاد خودکار محصولات جدید یا به‌روزرسانی موجود

## محصولات ساده

هر محصول ساده شامل:
```json
{
  "id": 123,
  "name": "نام محصول",
  "sku": "SKU123",
  "description": "توضیحات",
  "short_description": "خلاصه",
  "price": "50000",
  "stock_quantity": 10,
  "category": "الکترونیک",
  "type": "simple",
  "images": ["https://example.com/image1.jpg", "https://example.com/image2.jpg"]
}
```

## محصولات متغیر

محصولات متغیر شامل ویژگی‌ها و متغیرهای تفصیلی:
```json
{
  "id": 456,
  "name": "تی‌شرت",
  "sku": "TSHIRT",
  "price": "100000",
  "type": "variable",
  "attributes": {
    "رنگ": ["قرمز", "آبی", "سبز"],
    "سایز": ["S", "M", "L", "XL"]
  },
  "variations": [
    {
      "variation_id": 457,
      "sku": "TSHIRT-RED-S",
      "price": "100000",
      "stock_quantity": 5,
      "attributes": {
        "رنگ": "قرمز",
        "سایز": "S"
      },
      "image": "https://example.com/red-s.jpg"
    }
  ]
}
```

## نحوه استفاده

### 1. نصب پلاگین
- فایل `product-import-export.php` را در `wp-content/plugins/` قرار دهید
- در داشبورد وردپرس: افزونه‌ها → فعال‌سازی

### 2. دسترسی به ابزار
- تنظیمات ناحیه WooCommerce → Import/Export
- یا: WooCommerce → Import/Export

### 3. دانلود
- انتخاب فرمت (JSON/CSV)
- انتخاب نوع محصول (ساده/متغیر)
- کلیک بر "دانلود کنید"

### 4. آپلود
- انتخاب فایل JSON یا CSV
- کلیک بر "آپلود کنید"
- مشاهده پیشرفت و نتیجه

## نکات مهم

⚠️ **قبل از آپلود**:
- فایل را بک‌آپ کنید
- فرمت فایل را بررسی کنید
- محصولات موجود به‌روزرسانی می‌شوند

💡 **نکات سریع**:
- پلاگین به WooCommerce نیاز دارد
- فقط مدیران می‌توانند از این ابزار استفاده کنند
- قسمت فیلتر CSV برای انتقال سریع بین سایت‌ها بهترین است

## فرمت CSV

### سرستون‌های محصولات ساده
```
ID,نام,SKU,قیمت,موجودی,دسته‌بندی,نوع,توضیحات,عکس‌ها
123,محصول,SKU123,50000,10,دسته,simple,توضیح,url1;url2
```

### داده‌های متغیرهای محصول
برای هر متغیر (variation) باید یک ردیف جداگانه با `variation_id` داشته باشید

## مشکلات رایج

**Q: خطای "فایل JSON معتبر نیست"**
A: JSON فایل را با یک JSON validator بررسی کنید

**Q: محصولات آپلود نشده‌اند**
A: مطمئن شوید نام ستون‌ها صحیح است و دسترسی دارید

**Q: تصاویر دانلود نشدند**
A: پلاگین فقط URL عکس‌ها را دانلود می‌کند، نه خود فایل‌ها
