## ✨ ویژگی‌های اصلی

### 📥 دانلود محصولات
- ✅ فرمت JSON (برای محصولات متغیر)

- ✅ فیلتر محصولات ساده و متغیر
- ✅ تمام داده‌ها: نام، SKU، قیمت، موجودی، دسته‌بندی، عکس‌ها

### 📤 آپلود محصولات

- ✅ تشخیص و نمایش خطاها
- ✅ پشتیبانی از محصولات متغیر

### 🔒 امنیت
- ✅ نonce verification
- ✅ بررسی دسترسی‌ها
- ✅ Sanitization تمام ورودی‌ها
- ✅ فقط مدیران می‌توانند استفاده کنند

## 🚀 شروع سریع

### 1️⃣ نصب
```
1. فایل product-import-export.php را در wp-content/plugins/ قرار دهید
2. افزونه‌ها → فعال‌سازی
3. WooCommerce → Import/Export
```

### 2️⃣ دانلود
```
1. انتخاب فرمت (JSON/CSV)
2. انتخاب نوع محصول
3. کلیک "دانلود کنید"
```

### 3️⃣ آپلود
```
1. انتخاب فایل
2. کلیک "آپلود کنید"
3. منتظر تکمیل بمانید
```

## 📊 پشتیبانی محصولات

### محصول ساده
```
✓ نام
✓ SKU
✓ ID
✓ قیمت
✓ موجودی
✓ دسته‌بندی
✓ عکس اصلی + اضافی
✓ توضیحات
```

### محصول متغیر
```
✓ تمام مورد بالا + 
✓ ویژگی‌ها (Attributes)
✓ متغیرهای مختلف (Variations)
✓ قیمت برای هر متغیر
✓ موجودی برای هر متغیر
✓ عکس برای هر متغیر
```

## 🎯 نمونه استفاده

### دانلود JSON
```bash
1. داشبورد WooCommerce
2. Import/Export
3. فرمت: JSON
4. محصولات ساده + متغیر: ✓
5. دانلود
```

### تغییر در فایل
```json
{
  "id": 1,
  "name": "محصول",
  "sku": "SKU123",
  "price": "100000",
  "stock_quantity": 50
}
```

### آپلود تغییرات
```bash
1. فایل تغییر‌یافته را انتخاب کنید
2. آپلود
3. محصولات به‌روزرسانی می‌شوند
```
