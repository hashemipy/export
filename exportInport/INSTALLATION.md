# راهنمای نصب پلاگین محصول Import/Export

## الزامات
- WordPress 5.0+
- WooCommerce 3.0+
- PHP 7.4+

## مراحل نصب

### روش 1: از طریق cPanel (توصیه شده)

1. **فایل را آپلود کنید**:
   - به cPanel وارد شوید
   - فایل‌منیجر → wp-content → plugins
   - فایل `product-import-export.php` را درون پوشه `plugins` کپی کنید

2. **فعال‌سازی**:
   - داشبورد WordPress وارد شوید
   - افزونه‌ها (Plugins)
   - پلاگین "محصول Import/Export" را پیدا کنید
   - روی "فعال‌سازی" کلیک کنید

### روش 2: بارگذاری از داشبورد

1. افزونه‌ها → اضافه کردن جدید
2. فایل را بارگذاری کنید
3. فعال‌سازی

## استفاده از ابزار

### دسترسی
1. بروید: **WooCommerce** → **Import/Export**
2. یا: **تنظیمات** → **WooCommerce** → **Import/Export**

### دانلود محصولات

**مرحله 1**: انتخاب فرمت
```
○ JSON (پیشنهادی برای محصولات متغیر)
○ CSV  (برای جداول و برنامه‌های دیگر)
```

**مرحله 2**: انتخاب نوع‌ها
```
☑ محصولات ساده
☑ محصولات متغیر
```

**مرحله 3**: کلیک بر "دانلود کنید"

فایل به صورت خودکار دانلود می‌شود.

### آپلود محصولات

**مرحله 1**: فایل را انتخاب کنید
```
- فایل JSON یا CSV را انتخاب کنید
- فیلتر الگویی: *.json, *.csv
```

**مرحله 2**: کلیک بر "آپلود کنید"

**مرحله 3**: منتظر بمانید
```
- نوار پیشرفت نشان می‌دهد درصد
- بعد از تکمیل: پیام موفقیت یا خطا
```

## تعداد محصولات و کارآیی

| تعداد | سرعت | توصیه |
|------|------|--------|
| 1-100 | فوری | خیلی سریع ✓ |
| 100-500 | سریع | سریع ✓ |
| 500-1000 | 5-10 ثانیه | قابل قبول |
| 1000+ | 10+ ثانیه | ممکن است timeout شود |

### نکات برای فایل‌های بزرگ

1. فایل را به قسمت‌های کوچک‌تر تقسیم کنید
2. از یک اتصال انترنت پایدار استفاده کنید
3. timeout سرور را بررسی کنید (معمولاً 300 ثانیه)

## حل مشکلات

### ❌ "این فایل قابل پذیرش نیست"
**حل**: تنها فایل‌های `.json` یا `.csv` قبول می‌شوند

### ❌ "فایل JSON معتبر نیست"
**حل**: فایل را یک JSON Validator چک کنید
- https://jsonlint.com/

### ❌ "محصولات آپلود نشدند"
**مثال‌های احتمالی**:
- ستون‌های CSV نام‌ها اشتباه هستند
- JSON فایل بدفرمت است
- دسترسی مناسب ندارید

### ❌ صفحه خالی یا Error
**حل**:
1. WooCommerce فعال است؟
2. دسترسی مدیر دارید؟
3. فایل logs را بررسی کنید: `/wp-content/debug.log`

## نمونه فایل‌ها

دو نمونه فایل در پوشه موجود است:
- `sample-products.json` - فرمت JSON
- `sample-products.csv` - فرمت CSV

این فایل‌ها را می‌توانید برای تست استفاده کنید.

## بک‌آپ قبل از آپلود

⚠️ **همیشه**:
1. درخواست cPanel یا هاست
2. یک بک‌آپ کامل دانلود کنید
3. به‌طور اختیاری: فایل `wp_posts` و `wp_postmeta` را بک‌آپ کنید

## سوالات متداول

**Q: فایل بزرگ تر می‌توانم آپلود کنم؟**
A: حداکثر محدودیت `max_upload_size` سرور است (معمولاً 64MB-256MB)

**Q: آیا محصولات بدون نام می‌توانند ایجاد شوند؟**
A: خیر، نام الزامی است

**Q: آیا عکس‌ها خودکار دانلود می‌شوند؟**
A: خیر، فقط URL ذخیره می‌شود

**Q: آیا می‌توانم فقط موجودی به‌روزرسانی کنم؟**
A: بله، فایلی با تنها ID و موجودی بسازید

## تماس با پشتیبانی

اگر مشکل دارید:
1. فایل `debug.log` را بررسی کنید
2. سیاق خطا را کپی کنید
3. به هاست یا توسعه‌دهنده اطلاع دهید
