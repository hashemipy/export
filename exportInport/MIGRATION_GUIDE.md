# 📚 راهنمای مهاجرت محصولات بین سایت‌ها

این راهنما برای انتقال محصولات از یک سایت WooCommerce به سایت دیگر است.

## 📋 مراحل کلی

```
سایت مبدا (منبع)
     ↓
  دانلود (Export)
     ↓
  تغییر و ویرایش (اختیاری)
     ↓
سایت مقصد (هدف)
     ↓
  آپلود (Import)
     ↓
  تایید داده‌ها
```

## 🔧 آماده‌سازی

### سایت مبدا
```
1. پلاگین را نصب کنید (روی هر دو سایت)
2. نسخه WordPress و WooCommerce را بررسی کنید
3. یک بک‌آپ کامل بگیرید
```

### سایت مقصد
```
1. پلاگین را نصب کنید
2. WooCommerce را تنظیم کنید
3. ارز و واحدهای سنجش را یکسان کنید
```

## 🚀 مراحل دانلود (سایت منبع)

### مرحله 1: ورود به داشبورد
```
1. WordPress dashboard
2. WooCommerce
3. Import/Export
```

### مرحله 2: انتخاب گزینه‌ها
```
□ فرمت: JSON (توصیه)

```

### مرحله 3: دانلود

### مرحله 4: تایید
```

- هیچ خطایی نیست ✓
```

## 📝 ویرایش فایل (اختیاری)

### بررسی محتویات
```bash
# فایل را در notepad++ یا VS Code باز کنید
# یا از JSON Viewer استفاده کنید
```

### مثال: تغییر دسته‌بندی
```json
// قبل
"category": "الکترونیک"

// بعد
"category": "لوازم الکترونیکی"
```

### مثال: تغییر قیمت
```json
// قبل
"price": "100000"

// بعد
"price": "120000"  // 20% افزایش
```

### نکات ویرایشی
- ✅ نام را می‌توانید تغییر دهید
- ✅ قیمت را می‌توانید تغییر دهید
- ✅ موجودی را می‌توانید تغییر دهید
- ❌ ID را تغییر ندهید (مگر ID نباشد)
- ❌ SKU را تغییر ندهید (قبل پیدا کردن)

## 🔍 تایید داده‌ها

### بررسی ساختار
```
1. WooCommerce → محصولات
2. محصولات جدید را ببینید
3. کلیک بر یک محصول
4. تمام داده‌ها درست است ✓
```

### بررسی دسته‌بندی‌ها
```
1. WooCommerce → محصولات → دسته‌بندی‌ها
2. دسته‌بندی‌های جدید ظاهر شده است ✓
```

### بررسی عکس‌ها
```
1. کلیک بر محصول
2. عکس اصلی موجود است ✓
3. عکس‌های اضافی موجود هستند ✓
```

### بررسی متغیرها (اگر متغیر است)
```
1. محصول متغیر را باز کنید
2. تب "تغییرات" را کلیک کنید
3. تمام متغیرها موجود هستند ✓
4. قیمت و موجودی درست است ✓
```

## 🔧 حل مشکلات

### ❌ "محصولات آپلود نشدند"

**علت احتمالی**:
1. نام ستون‌های CSV اشتباه
2. فایل JSON بدفرمت
3. دسترسی محدود

**حل**:
```
1. فایل را دوبارہ بررسی کنید
2. JSON Validator استفاده کنید
3. پشتیبانی را تماس بگیرید
```

### ❌ "Timeout شد"

**علت احتمالی**:
1. فایل خیلی بزرگ
2. اتصال اینترنت ضعیف
3. سرور آهسته

**حل**:
```
1. فایل را به دسته‌های کوچک‌تر تقسیم کنید
2. اتصال قوی‌تر استفاده کنید
3. ساعات کم‌ترافیک را انتخاب کنید
```

### ❌ "عکس‌ها دانلود نشدند"

**علت احتمالی**:
1. URL معتبر نیست
2. سرور عکس خاموش است
3. اجازه دسترسی نیست

**حل**:
```
1. URL‌ها را بررسی کنید
2. اتصال سرور را تست کنید
3. مجوزهای دسترسی را بررسی کنید
```
