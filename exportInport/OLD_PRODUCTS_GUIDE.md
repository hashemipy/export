# راهنمای محصولات قدیمی

## مشکل محصولات قدیمی

محصولاتی که خیلی وقت است ایجاد شده‌اند، ممکن است:

1. **بعضی متغیرات حذف شده باشند**
   - اما ویژگی‌های اصلی موجود هستند

2. **بعضی متغیرات قیمت ندارند**
   - قیمت محصول اصلی استفاده می‌شود

3. **بعضی متغیرات موجودی ندارند**
   - صفر موجودی ثبت می‌شود

## حل سازمان‌یافته

### مرحله 1: دانلود

متغیرات **حتی اگر خالی باشند** دانلود می‌شوند:

```
✓ متغیرات پاک‌شده‌ای در فایل JSON وجود دارد
✓ اگر قیمت ندارند، خالی است
✓ اگر attributes ندارند، خالی است
```

### مرحله 2: بررسی

**نتایج:**
```
✓ محصول معتبر است (حتی اگر متغیرات ناقص باشند)

⚠️ هشدارها:
• متغیر 1: قیمت موجود نیست (قیمت محصول استفاده می‌شود)
• متغیر 1: ویژگی‌های متغیر خالی است (فقط ایجاد می‌شود)
```

→ **فقط هشدار است** (نه خطا)، می‌توانید آپلود کنید

### مرحله 3: آپلود

**چه اتفاقی می‌افتد:**
1. محصول ایجاد می‌شود
2. متغیرات ایجاد می‌شوند (حتی بدون attributes)
3. اگر قیمت خالی بود، قیمت محصول استفاده می‌شود
4. موجودی 0 یا مقدار موجود استفاده می‌شود

## مثال

### محصول دانلود شده:

```json
{
  "name": "تیشرت شلوارک",
  "price": "865000",
  "type": "variable",
  "attributes": {
    "color": {
      "values": [
        {"name": "سبز", "slug": "green"},
        {"name": "سفید", "slug": "white"}
      ]
    }
  },
  "variations": [
    {
      "sku": "",
      "price": "",
      "stock_quantity": 0,
      "attributes": []
    },
    {
      "sku": "",
      "price": "",
      "stock_quantity": 0,
      "attributes": []
    }
  ]
}
```

### نتیجه بررسی:

```
✓ 1/1 محصول معتبر

⚠️ هشدارها (4):
• محصول "تیشرت شلوارک" - متغیر 1: قیمت موجود نیست
• محصول "تیشرت شلوارک" - متغیر 1: ویژگی‌های متغیر خالی است
• محصول "تیشرت شلوارک" - متغیر 2: قیمت موجود نیست
• محصول "تیشرت شلوارک" - متغیر 2: ویژگی‌های متغیر خالی است

✓ فایل معتبر است! می‌توانید اپلود کنید
```

→ دکمه "آپلود" فعال است

### نتیجه آپلود:

```
✓ محصولات: 1 | متغیرات: 2 | ...

متغیرات ایجاد شدند:
• متغیر 1: قیمت محصول (865000) + stock = 0
• متغیر 2: قیمت محصول (865000) + stock = 0
```

## نکات مهم

✓ **Variations خالی اوکی است**
  - ایجاد می‌شوند
  - بعد می‌توانید ویژگی‌ها را اضافه کنید

✓ **قیمت خالی اوکی است**
  - قیمت محصول استفاده می‌شود
  - بعد می‌توانید تغییر دهید

✓ **موجودی خالی اوکی است**
  - صفر ثبت می‌شود
  - بعد می‌توانید آپدیت کنید

## مراحل بعد از آپلود

1. برروید WooCommerce → محصولات
2. محصول را باز کنید
3. متغیرات را بررسی کنید
4. ویژگی‌ها را اضافه کنید (اگر لازم)
5. قیمت‌ها را آپدیت کنید (اگر لازم)
6. موجودی را آپدیت کنید

## مثال عملی

### قبل:
```
محصول قدیمی
├─ متغیر 1 (حذف شده)
├─ متغیر 2 (حذف شده)
└─ متغیر 3 (حذف شده)
```

### بعد از انتقال:
```
محصول جدید
├─ متغیر 1 (ایجاد شد - صفر موجودی)
├─ متغیر 2 (ایجاد شد - صفر موجودی)
└─ متغیر 3 (ایجاد شد - صفر موجودی)
```

### بعد از آپدیت:
```
محصول جدید
├─ متغیر 1 (ویژگی‌ها اضافه شد - موجودی: 5)
├─ متغیر 2 (ویژگی‌ها اضافه شد - موجودی: 3)
└─ متغیر 3 (ویژگی‌ها اضافه شد - موجودی: 0)
```
