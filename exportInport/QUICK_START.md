# ⚡ شروع سریع - 5 دقیقه

## 🎯 هدف
دانلود و آپلود محصولات بین سایت‌های WooCommerce بدون پیچیدگی

## 📦 فایل‌های پروژه

```
✅ product-import-export.php       (خود پلاگین - 18 کیلوبایت)
✅ README.md                       (مقدمه)
✅ QUICK_START.md                  (این فایل)
📖 PLUGIN_README.md                (راهنمای کامل)
📖 INSTALLATION.md                 (نصب و تنظیم)
📖 MIGRATION_GUIDE.md              (انتقال محصولات)
👨‍💻 DEVELOPER.md                   (برای توسعه‌دهندگان)
📄 sample-products.json            (نمونه JSON)
📄 sample-products.csv             (نمونه CSV)
```

## ⏱️ نصب - 1 دقیقه

### گام 1: آپلود فایل
```
از طریق cPanel:
1. فایل‌منیجر
2. wp-content/plugins/
3. فایل product-import-export.php را درون آن قرار دهید
```

### گام 2: فعال‌سازی
```
داشبورد WordPress:
1. افزونه‌ها
2. "محصول Import/Export" را پیدا کنید
3. "فعال‌سازی"
```

### گام 3: تایید
```
1. WooCommerce → Import/Export
2. صفحه ظاهر شد ✓
```

## ⏱️ دانلود محصولات - 2 دقیقه

```
WooCommerce → Import/Export

📥 قسمت دانلود:
  ○ JSON (توصیه)
  ○ CSV
  ☑ محصولات ساده
  ☑ محصولات متغیر
  [دانلود کنید]

فایل دانلود می‌شود! ✓
```

## ⏱️ آپلود محصولات - 2 دقیقه

```
WooCommerce → Import/Export

📤 قسمت آپلود:
  [انتخاب فایل...]
  [آپلود کنید]
  
نوار پیشرفت → ✓ تکمیل
```

## ❓ سوالات فوری

**Q: چند محصول می‌توانم آپلود کنم؟**
A: 1-500 بهترین است. بیش‌تر از آن را تقسیم کنید.

**Q: فایل را کجا دانلود می‌کنم؟**
A: مرورگر خودکار دانلود می‌کند.

**Q: عکس‌ها کپی می‌شوند؟**
A: فقط URL‌ها. عکس‌های جدید خودکار دانلود نمی‌شوند.

**Q: محصولات قدیمی حذف می‌شوند؟**
A: خیر، نو محصول‌ها فقط اضافه می‌شوند.

**Q: SKU تکراری چه می‌شود؟**
A: محصول موجود به‌روزرسانی می‌شود.
